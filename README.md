# Support Desk

Support desk is a support ticket app built with MERN stack

# Technologies Used

- Nodejs/Express
- MongoDB
- React
- Redux
- JWT for authentication

# Install and Run

- Create a .env file and add your `MONGO_URI` and `JWT_SECRET`

## Install backend dependencies

```
npm install
```

## Install frontend dependencies

```
cd frontend
npm install
```

## Run app (development)

```
npm run dev
```
